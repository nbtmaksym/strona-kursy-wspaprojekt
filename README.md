# LearnUp 🎓

Platforma kursów online — projekt portfolio.

## Technologie

**Frontend**
- HTML5 (semantyczny)
- CSS3 (Flexbox, Grid, Custom Properties, animacje)
- JavaScript (Vanilla JS, Intersection Observer, walidacja)

**Backend** *(Commit 2)*
- Python + FastAPI
- SQLAlchemy + SQLite
- JWT (autoryzacja)
- bcrypt (hashowanie haseł)

## Strony

| Plik | Opis |
|------|------|
| `frontend/index.html` | Strona główna (landing page) |
| `frontend/kursy.html` | Lista kursów z filtrowaniem |
| `frontend/login.html` | Logowanie |
| `frontend/register.html` | Rejestracja |
| `frontend/dashboard.html` | Panel użytkownika |
| `frontend/admin/index.html` | Panel admina statystyki |
| `frontend/admin/kursy.html` | Panel admina zarządzanie kursami |
| `frontend/admin/uzytkownicy.html` | Panel admina — użytkownicy |

## Funkcjonalności

- Responsywny layout (mobile-first, 2 breakpointy)
- Tryb jasny / ciemny
- Animacje przy scrollowaniu (Intersection Observer)
- Menu hamburger na mobile
- Walidacja formularzy
- Filtrowanie kursów po poziomie i kategorii
- Panel admina: dodawanie, usuwanie, wyszukiwanie kursów i użytkowników

## Uruchomienie

Otwórz `frontend/index.html` w przeglądarce lub użyj Live Server w VSCode.
